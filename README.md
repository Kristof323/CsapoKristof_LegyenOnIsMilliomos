# 🌟 Legyen Ön is Milliomós!

C# nyelven írt kvizjáték, a népszerű TV-műsor alapján.

## Leírás

Ez a projekt a "Legyen Ön is Milliomós!" kvizjátékot valósítja meg C# nyelven. A játékos kérdésekre válaszol, és egyre nagyobb nyereyményeket érhet el.

## Funkciók

- Több félevés (puska) használata
- Fokozatosan nehézedő kérdések
- Pontrendszer és nyereménylétra

## Technológiák

- **Nyelv:** C#
- **.NET** keretrendszer

## Használat

1. Klónozd a repót:
   ```bash
   git clone https://github.com/Kristof323/CsapoKristof_LegyenOnIsMilliomos.git
   ```
2. Nyisd meg Visual Studio-ban
3. Fordítsd és futtasd
